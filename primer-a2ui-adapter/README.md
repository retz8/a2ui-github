# primer-a2ui-adapter

The publishable A2UI catalog/adapter mapping A2UI components to [Primer](https://primer.style/)
React wrappers, built over [`@a2ui/react`](https://www.npmjs.com/package/@a2ui/react).

> 🚧 **Skeleton.** This package is currently a placeholder — `catalog.ts`, `catalog-id.ts`, and
> `catalogs/v0.9.1/catalog.json` are stubs, and `components/` is empty. The real catalog (Primer
> wrappers and the `Catalog` built over the v0.9 basic catalog) is wired in a later phase.

## Commands

```bash
yarn workspace primer-a2ui-adapter run build      # tsc → dist/
yarn workspace primer-a2ui-adapter run typecheck
yarn workspace primer-a2ui-adapter run test       # vitest
yarn workspace primer-a2ui-adapter run lint
```
